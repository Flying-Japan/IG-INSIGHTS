# IG 인사이트 대시보드 — 제작 일지

> **프로젝트**: Flying Japan 인스타그램 계정 데이터 분석 대시보드
> **대상 계정**: 일본 여행 인프라 정보 (한국 타겟)
> **배포 URL**: https://flying-japan.github.io/IG-INSIGHTS/
> **기술 스택**: HTML/CSS/JS (Vanilla), Tabulator 6.3.0, ApexCharts, SheetJS

---

## 📋 기능 적용 목록

### 1. KPI 통합 대시보드 (개요 탭)
| 항목 | 내용 |
|------|------|
| **기능** | 게시물 수, 팔로워, 도달, 조회수, 좋아요, 저장, 공유, 댓글, 참여, 참여율, 저장율, 공유율, 팔로우 유입, TOP 게시물 — 14개 KPI 카드 |
| **모드** | 합계 / 평균 / 년도별 / 월별 / 주별 / 일별 드롭다운 전환 |
| **목적** | 한 눈에 계정 전체 성과를 파악할 수 있는 종합 뷰. 기간별 모드로 특정 기간의 성과를 분석 가능 |
| **분석 활용** | "이번 달 도달이 전월 대비 어떻게 변했는가", "평균 참여율 추이가 어떤가" 등 보고서 기반 데이터 제공 |

### 2. 전일 대비 변화량 표시 (초록/빨간 배지)
| 항목 | 내용 |
|------|------|
| **기능** | 각 KPI 카드에 전일 대비 변화량을 +/- 배지로 표시 (초록: 증가, 빨간: 감소) |
| **데이터** | `daily_report.json`의 최근 2일간 데이터 비교 |
| **목적** | 일일 성과 변동을 실시간으로 감지하여 빠른 대응 가능 |
| **분석 활용** | "어제 도달이 +5,759 증가했으니 특정 게시물이 알고리즘에 탔을 가능성", "저장 +132 증가는 정보성 콘텐츠 반응" |

### 3. 벤치마크 등급 시스템 (우수/양호/보통/미흡)
| 항목 | 내용 |
|------|------|
| **기능** | 참여율, 저장율, 공유율, 도달율 등에 여행 업종 2025 기준 등급 배지 표시 |
| **등급** | 우수 (초록) → 양호 (파랑) → 보통 (노랑) → 미흡 (빨강) |
| **목적** | 단순 수치가 아닌 업종 대비 상대적 위치를 파악하여 "잘 하고 있는가"를 즉시 판단 |
| **분석 활용** | "참여율 5.8%로 우수 등급 — 업종 평균(2~3%) 대비 2배 이상", "공유율은 양호로 개선 여지 있음" |

### 4. 툴팁 설명 + 벤치마크 스케일
| 항목 | 내용 |
|------|------|
| **기능** | 각 지표 옆 ⓘ 아이콘에 마우스를 올리면 지표 설명 + 여행 업종 기준 척도 바 표시 |
| **목적** | 비전문가도 각 수치의 의미와 기준을 이해할 수 있도록 정보 제공 |
| **분석 활용** | 보고 시 "이 수치가 왜 중요한지", "업종 평균 대비 어디에 위치하는지" 근거 제공 |

### 5. 운영 기점 필터 (담당 이전/이후)
| 항목 | 내용 |
|------|------|

| **목적** | 담당자 변경 전후의 성과를 분리하여 운영 효과를 정량적으로 비교 |
| **분석 활용** | "담당 이후 평균 참여율이 3%→5.8%로 상승", "담당 이후 기간에 저장율이 개선" 등 성과 입증 |

### 6. 운영 기여도 분석 (도넛 차트)
| 항목 | 내용 |
|------|------|
| **기능** | 전체 누적 데이터 중 담당 이후 기간의 기여 비중을 도넛 차트로 시각화 |
| **지표** | 도달, 참여, 저장, 팔로우 유입 등 주요 지표별 기여율 |

| **분석 활용** | 짧은 운영 기간에도 높은 기여율을 달성했다면 운영 효율성 입증 |

### 9. 게시물 테이블 (Tabulator 6.3.0)
| 항목 | 내용 |
|------|------|
| **기능** | 순위, 업로드일, 유형, 카테고리, 제목, 도달, 조회수, 좋아요, 저장, 공유, 댓글, 팔로우, 전환율, 참여율, 점수 |
| **정렬** | 드롭다운 14종 (종합순위, 최신순, 조회수순 등) + 헤더 클릭 정렬 |

### 12. 숫자 표시 토글 (K/M ↔ 전체 숫자)
| 항목 | 내용 |
|------|------|

| **적용 범위** | 게시물 테이블, KPI 카드, 카테고리 테이블, TOP10 테이블, 변화량 배지 |

| **목적** | 정밀 분석 시에는 전체 숫자, 빠른 파악 시에는 K/M 축약으로 전환 |
| **분석 활용** | 보고서 작성 시 정확한 수치 확인, 일상 모니터링 시 한눈에 파악 |

### 13. 정렬 순서 동기화 (헤더 클릭 + 드롭다운)
| 항목 | 내용 |
|------|------|
| **기능** | Tabulator 헤더 클릭 정렬 시 순위(_rank)가 자동 동기화. 드롭다운 전환 시 내부 소터 클리어 |
| **기술** | `dataSorted` 이벤트 + debounce 50ms + `clearSort()` |
| **목적** | 어떤 방식으로 정렬하든 순위 표시가 항상 정확하게 반영 |

### 14. 날짜 정렬 정확도 개선
